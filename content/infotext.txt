Welcome to the Unknown Horizons Alpha 2009.01, Keybindings:
--------------
- LEFT = Scroll Left
- RIGHT = Scroll Right
- UP = Scroll Up
- DOWN = Scroll Down
- "," = Rotate Building Left
- "." = Rotate Building Right
- U = Show Build Menu
- F1 = Display Help
- F5 = Quicksave
- F9 = Quickload
- F10 = Toggle console on / off
- ESC = Show Pause menu
- T = Toggles grid on / off
- X = Enable destruct mode
- P = Pause
- "+" = Speed Up
- "-" = Speed Down
- S = Screenshot
- B = Launch Python Debugger


Have fun.
